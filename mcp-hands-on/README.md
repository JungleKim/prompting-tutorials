# MCP 핸즈온 워크샵

이 저장소는 Model Context Protocol(MCP)를 활용하는 다양한 예제 애플리케이션을 포함한 핸즈온 워크샵입니다.

## 소개

Model Context Protocol(MCP)은 대규모 언어 모델(LLM)이 외부 도구와 안전하게 상호작용할 수 있게 해주는 프로토콜입니다. 이 워크샵은 MCP의 다양한 활용 방법을 실습할 수 있는 예제들을 제공합니다.

## 예제 목록

### 1. 날씨 정보 조회 애플리케이션 (`/weather`)

미국 기상청(NWS) API를 활용하여 날씨 정보를 조회하는 MCP 애플리케이션입니다.

**주요 기능:**
- 미국 주별 날씨 경보 조회 (`get_alerts`)
- 위도/경도 기반 날씨 예보 조회 (`get_forecast`)

[날씨 애플리케이션 바로가기](/weather)

### 2. 커머스 데이터 분석 애플리케이션 (`/commerce-analytics`)

커머스 서비스의 판매 데이터를 분석하는 MCP 애플리케이션입니다.

**주요 기능:**
- 기간별 매출 분석
- 상품 카테고리별 매출 분석
- 고객 세그먼트 분석
- SQL 쿼리 실행 및 시각화
- 데이터 기반 리포트 생성

[커머스 데이터 분석 애플리케이션 바로가기](/commerce-analytics)
